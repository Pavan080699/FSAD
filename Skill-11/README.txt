SKILL 11 - React API Integration inside Spring Boot

Package Name: com.klu.skill11
Artifact ID: skill11app
Java Version: 17
Spring Boot: 3.x

How to run in Eclipse:
1. Open Eclipse / Spring Tool Suite.
2. Import as Existing Maven Project.
3. Select the folder com.klu.skill11.
4. Wait for Maven dependencies to download.
5. Run Skill11Application.java as Java Application.
6. Open browser: http://localhost:8080

Included features:
- Part A: Local users from users.json using fetch()
- Part B: JSONPlaceholder users using fetch()
- Part C: DummyJSON posts using axios
- Refresh button in Fake API Posts
- Dropdown filter by userId
- Dashboard navigation with buttons
- Basic CSS styling

Note:
This is a Spring Boot project serving a React app from static/index.html using CDN-based React and Babel, so it runs directly in Eclipse without separate Node.js setup.
