SKILL 12 - Full Stack CRUD Application using React and Spring Boot

Package Name: com.klu.skill12
Artifact ID: studentcrudapp
Java Version: 17
Spring Boot: 3.x

Features completed:
1. POST /students
2. GET /students
3. PUT /students/{id}
4. DELETE /students/{id}
5. Service layer and Repository layer
6. ResponseEntity used in controller
7. React frontend with axios
8. Add, Update, Delete, and auto-refresh student list
9. Form clear after submission
10. Basic CSS styling

Run in Eclipse:
1. Extract the ZIP.
2. Open Eclipse or Spring Tool Suite.
3. Import -> Existing Maven Project.
4. Select folder: com.klu.skill12
5. Wait for Maven build.
6. Run Skill12Application.java
7. Open http://localhost:8080

Project note:
This version uses in-memory storage in the repository, so no database setup is required. It is easy to demonstrate in lab and works directly in Eclipse.
